「用尽我所有的力气，热爱生活中的一切」-- 我的 2019

发表于 2020-01-21 | 分类于 生活随笔 | | 阅读次数:

我去年的年终总结写的很昏暗很沉重，但现在回看当初为自己立下的目标基本都还是实现了。这些目标当初定下的时候或许我自己都没有太想明白，我真正想要的是什么。但经过了这一年生活的“重新开始”，给自己一个能说服自己的答案应该是没问题了。

先盘点一下年初的目标

  • 买车
    • 买车这个目标算是完成的比较早的，说真的要不是因为在年初早早的就把车搞定，我这一年的生活未必能这么潇洒自在，未必能真正意义上做到我昵称里的那个形容词“折腾范儿”，变成了一个周六在家就闲不住星人，周日不出门折腾会死星人～
    • 很多朋友说我买这个车其实是买了个大玩具，确实如此。除了日常用车需求之外，围绕着这个车的各种科技与性能本身，给我的生活带来了极大的乐趣，即便伴随而来的大额车贷负担，也无法抵挡“真香”的魔力（@鱼雷导弹 开完笑说这是车才是我的电动“媳妇儿”）
    • Tesla Model S 的亲密接触
    • Tesla Model S 长途体验与自动驾驶
  • 装修
    • 当初给自己制定重新装修屋子的目标其实是希望让自己与过去的生活告个别。过去的我太不在意生活小节，沉浸在自己的技术兴趣与游戏乐趣之中(或许也伤害了别人)。定下这个目标希望能以装修作为一个出发点，思考一下我的“生活”应该是怎样的状态才算健康？
    • 这个目标现在看也算是完成的相当不错，不仅把重新收拾厨房和卫生间吊顶管道与电路这件遗留很久的历史问题解决。原本规划只作为尝试自己心里没太大谱的智能家居，也算是超预期完成。几乎全家 90% 电器都可以智能控制，不仅仅是主动的远程/语音控制，还可以无需任何命令的全自动的让家中电器根据屋内人员状态进行自动控制调节
    • 坦白说，智能家居很多东西说他是玩具不为过，就算做到 90% 家电的智能，现阶段这种智能有还是没有，都不会对生活有太大影响，但这个过程中让我把兴趣与精力投入到“经营”生活中去，并且真的感受到舒适与快乐，还是挺美好的一件事情。一点点的小改动，让自己永远告别出家门忘记关灯的困扰，回到家后开门自动亮灯的迎接，都会让我觉得这个冰冷的家因为我的小举措有了不一样的令人开心愉悦的变化
    • DIY 你的智能管家 J.A.R.V.I.S – 米家 “智能家居” 深度改造
  • 锻炼
    • 这个目标简单而干脆，完成的也算不赖，我的本意不是为了减肥而减肥，而是希望锻炼自己的自制力，不求健身塑形，但求坚持留下好身体与优良的生活习惯。慢慢的，每日早饭，控制饮食，戒除零食，坚持锻炼。这一年下来，算是养成了一定的生活习惯了
    • 年初立下目标的时候，体重大概145斤不到，现在135斤不到，瘦了10斤了。其实这个成果到了年中 6～7 月份的时候就已经差不多了，没有再进一步的调整目标，在原有的日常习惯下保持了下来
  • 旅行
    • 以前的我也是每年会四处旅行，出国浪，但今年的旅行和过去都不太一样了。自己规划行程和甩手掌柜相比，那真是收获更多的快乐与回忆！一个人自己出去浪更是自由到 High ～不过很开心，我每次出去旅游写游记的习惯依旧保持了下来，用文字与照片记录下来的回忆，每次翻看都是非常的感动，哪怕时光飞逝，物是人非
    • 年初定的出国游保一争二的目标算是圆满完成，最想去的红叶季日本京都，伴随着充分的规划准备，开心痛快的浪回来了。唯一遗憾的是差一点点就实现了美帝之旅，要不是 sb 唐建国的贸易战，我的美签也不会被 check，导致耽误了 WWDC 没能公派出差成功
    • 为了弥补美帝的遗憾，国内游我也没少折腾：1）组织策划整个团队 20 来人的内蒙草原自驾游。2）全权安排规划了小伙伴一行4人的北疆深度游
    • 「一路向北」– 草原避暑自驾游记
    • 「金秋喀纳斯，神の后花园」– 秋日北疆深度自驾游记
    • 「京都归来不看枫」– 秋日关西漫步
  • 收入
    • 坦白说，这一条是唯一没完成的年初目标
    • 年初的目标是希望自己能开拓思维，攒or挣更多的钱。但今年，我比往常花的钱更多，攒下来的少了很多。并且我的精力与关注点都在“折腾”与“浪”上了，更没有多少能分配给“理财”或者“外块”之上
    • 所以关于这一点，我还是要反思的，其实主要的原因还是买车有点伤 =。=
我曾经一度认为自己生活挺失败的，想要证明自己



想让自己活出不一样的自己，告别过去，去热爱未来生活中的一切


从装修中，我确实再一步一步感受生活
# 扯淡
