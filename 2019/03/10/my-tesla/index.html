购车感受

北京操蛋的车牌政策

由于北京燃油车指标摇号常年不中，已经积累到了 5 倍概率，依然看不到中号的希望，几千分之一即便乘以 5 依然还是遥不可及，于是我在 2016 年中切换成了北京新能源指标，终于在一年多之后自然的“排队”排到了新能源号牌，号牌有效期1年。

18年上半年的北京车展也特意去逛了新能源专场，虽然 Tesla 看起来是压根不屑于参加北京车展，没看到展台，但也还是了解了非常多的其他厂家的。由于北京的新能源牌照只认纯电不认混合，在纯电的厂商里，当时看起来比较好的就是“比亚迪”和“荣威”了。另外还看到很多北汽的超低价5W-7W不等的占号车。

18年下半年，我也专门走访了比亚迪，宝马，特斯拉的销售门店，现场深入摸摸车深入的进行了解.

我当时的电动车选择心态是

  • 北汽垃圾车：
    • 死也不想买～
    • 6W块钱左右基本等于白扔，用车那个里程与续航就是扯淡，就算不用还得天天发愁停车，我还不如每年花 2000 多左右，去淘宝办理指标延期
    • 我对北汽这个在北京地方保护政策下的企业充满了鄙视，觉得这种企业就该死，就是靠吸国家补贴的蛀虫，更何况现在国家补贴逐渐在减少
  • 新能源创新车企：
    • 在18年的北京车展上，那会 蔚来/小鹏/理想 什么的好像都没怎么看到 ～
    • 对外的宣传也不多，18年上半年的时候完全没概念，似乎下半年才突然冒出来很多
  • 比亚迪/荣威：
    • 还是挺有吸引力的，但还是不太想买～
    • 价位 10-40W 的各个区间都有，各方面感觉看起来还是挺不错的
    • 北京号牌如此之艰难，车买下来想买个稍微好点的，想上个 BBA 级别中档点的
  • 宝马:
    • BBA 中唯一有纯电车的，i3/i8～
    • i8 那接近 200W 的价格就算了吧 =。=
    • i3 伴随着 4S 店的各种优惠之下，能用 30+ 拿下
    • i3 实在是太丑了，丑到不能忍，虽然是2排座位，但空间看着感觉没比 QQ 大多少
    • 宝马 530 的混动真的很吸引人，但是北京的新能源不认混动，没辙
  • 特斯拉/拓速乐：
    • Model S/X 的价位一下超出宝马 530Le 不少，预算压力还是挺大的
    • Model 3 之前一直是预定不知道猴年马月能拿车，但价格非常有吸引力
    • Model S/X 太好看了尤其是 S，但 3 的外形就觉得有一点丑了
    • 我对 SUV 车型没那么感冒，城市通行还是想要舒适的轿车
    • 去销售门店试驾，感受驾驶体验，无论是车辆召唤，动能回收，还是自动驾驶，感觉真的是太好了，欲罢不能

从上面对于纯电动车的选择心态上可以看出来，我已经非常非常偏向特斯拉了，不过我还是有个看法，我不是啥特斯拉吹，或者电动吹，我始终认为如果新能源指标能买混动，绝对死也不买电动爹，但我身在北京，就是这么的无奈，唉～

曲折下单的心路历程

因为生活与工作的原因，2018 年下半年，我换到了国贸上班，从我自己家回龙观到国贸的通勤变得很不方便。买车的心思越来越重，BBA 的 4S 店我也没少逛，在确认宝马奔驰奥迪的纯电动车新车型遥遥无期，毫无可盼的情况下，就开始更高频率的关注特斯拉的各方面新闻，也专门跑去特斯拉的直营销售店去试驾与体验。

本来 Model S 的最低配 75D，大概 80W 能下来，就是一直觉得超出预算挺多的，一直没下定决心。

2018年7月，突如其来的贸易战让美国进口车关税从25%提升到40%，于是特斯拉的价格直接涨了20W，丐版 Model S 也得 100W 才能拿下，彻底浇灭了我蠢蠢欲动的欲望。

2018年11月，随着贸易战你来我往打了那么久，似乎 Elon Musk 马一龙老铁察觉到了贸易战即将松动缓和的信号，官方宣布补贴关税差价导致的价格上涨，75D 又回到了 80W 能拿下来的价格。并且官方调整了配置清单，把皮质座椅/全景天窗/木制内饰材质/高级音响系统/生物武器防御级别的空气净化等全都做成了标配不在加价选配，相当于加量不加价，更加的划算了一些，于是我又开始蠢蠢欲动了。

2018年12月，贸易战在汽车关税上，两国政府宣布进入4个月的停战期，关税恢复正常水平到3月31日（这个时间很关键），马一龙老铁又在原本官方补贴关税，不到 80W 左右拿下 75D 的基础之上，又降价了几万块，标配裸车 74W 左右拿下 Model S 75D，我心里更加的躁动了。

并且18年底，传说中的 Model 3 已经离进入中国越来越近了，官方全面开放了后驱长续航/双电机/双电机性能版三款车型，并且销售统一口径，3月底肯定能提现车，现在可以赶紧下订单。3月底这个时间恰恰正是关税停战期的 Deadline ～ 并且放出了最新的消息，特斯拉要在上海建立工厂，专门组装生产 Model 3（不产S/X）～ 不过 Model 3 现在公布的这三款下来，算上购置税等落地也得 50-70W 不等看配置，真正让 Model 3 值得期待的是还没面世，就是 3.5W 美元那款低配，上海工厂生产的话估计能进 30W 的那款。而且 Model 3 亲自去看了一下，空间真的比 S 小了很多，没有仪表盘感觉特别特别的不习惯。

2019年1月初，马一龙老铁在推特上又一鸣惊人了，说从 1.14 周一起停售 75D，就只有高配 100D 了卖了，得活脱脱贵出 10 多W。这尼玛我一下坐不住了，之前心动 S 已经很久了就是实在是有点超出预算，因此一直在犹豫，这一下好了，停售就彻彻底底的更加超出预算彻底没戏了。

头像是新垣结衣的销售小姐姐赶紧在给我计算更加便宜划算的买法。现在因为 Model S 如果是预定车一定会标配皮质座椅/木制内饰材质，不用额外选配加钱。但现在已经入关的现车里刚好就有，半皮半织物座椅+黑色烤漆内饰的现车，并且是我最想要的红色车漆。（说真的烤漆内饰简直是意外惊喜，因为车内的木制装饰颜色看起来太老气了，换成烤漆反而感觉更加好看的多）

  • 皮质座椅/木制内饰本为标配，因此买这台现车反而会给我优惠 -35500
  • 因为选的红色车漆需要 +23500

这一加一减，销售小姐姐算了一下 73W 能拿下我最想要的红色 Model S ，我这个心动啊，我这个纠结啊，最终下定决心，在 1.18 的这个周五，直接交了定金下了订单。而下一周就停售了（事实证明，马一龙说话也有扯淡的时候，现在 75D 改了个名字又 TM 重新开卖了）

令人惊讶的提车速度

本来 1.11 下的订单，按着国内四儿子店的尿性，我心想怎么不得 2.10 春节假期后提车来。谁成想特斯拉这种没有4S的直营销售模式，公司要求现车必须在5个工作日内交付，提车速度快到令人发指！我刚好下一周在杭州出差，谁成想一周的时间就可以提车了。。。

我的新能源资格本来在 2019.02.26 到期，但我确实没想到能这么快突然就下定决心订车，也没想到特斯拉提车这么快速，我早在12月就办好了电动车延期，淘宝花了2300代办。这下这个延期彻底白办了 ╮(╯_╰)╭

  • 1.11 周五晚上下的订单
  • 1.12 听说我周日要飞机飞杭州，在周六就联系好招行的贷款资质认证
  • 1.15 招行车贷批准通过，在杭州转账，结清首付尾款，缴纳人保保险
  • 1.16 招行车贷到账，我可以提车了，然而我人还在杭州
  • 1.19 周六，我刚从杭州回来就能提车了，临牌也办好了
  • 1.23 周三，请了天假上好了车牌

特斯拉交付中心在亦庄，我本来计划在大清早约朋友赶往亦庄提车，但是交付中心的小姐姐说，现在国内也在推广美国的那种送车上门的提车模式，我可以成为首批体验这种服务的车主～那必须体验啊～周末中午，交付小姐姐开着我的红色 Model S 送车上门啦～

特斯拉还赠送了全套上牌服务，帮我提前办理各种上牌手续，只等一天亲自跑到五方桥现场验车选牌。北京新能源的号牌允许在网上选号，可以自己输入自己喜欢的号进行尝试，因为新能源号牌还很空，应该能比较容易的选到心仪的号码。但是网上选好不一定当天能出，可能还要再拖一天来拿号牌上牌。之前的年假已经被我去日本用的快光了，所以额外多请一天假对我来说太奢侈了，硬着头皮现场摇号吧！随机五十个号选一，走你！一发入魂！

本来看着这随机的五十个号我正看花眼呢，周围围观的群众一眼就帮我指出来这里面看着最牛逼的号，AD29800，都在说我这手气简直太好了，得嘞，就它了！

用车体验

驾驶与操控体验

电动车在驾驶的感觉上真的和以前开燃油车的感觉太不一样了，简直太舒服太好开了～

  • 线性加速：电动发动机的加速是线性的，油门轻轻踩下去就能直接有很明显的速度感。不像燃油发动机那样用力踩下去转速彪到3～4千，速度也是逐渐加起来的那种感觉。所以在堵车路段，电动车快速起步钻进钻出有很大的优势～

  • 推背感：Model S 75D 的百公里加速是 4.4 秒，如果上 P100D 能到 2.9 秒。查了一下官网，保时捷 Boxster 是 5.6 秒，Boxster GTS 4.1 秒。不扯那么多数字，就说一下真实感受，推背感这个词可以换一个字，“踹”背感 ╮(╯_╰)╭ 夜深人静找个完全没人的路上，一脚“电”门踩到底，一发地板“电”，那个感觉简直太夸张了，把你按在座椅上的感觉～顺带说一下，电动发动机在加速这块有着很大的优势，很多很多的国内新能源车型都能把百公里加速这个指标拉到 5 秒以内。

  • 动能回收：动能回收的驾驶体验简直太好了！所谓动能回收就是，当驾驶员轻抬油门的时候，就会自动触发刹车制动，并且回收一部分电能。而当驾驶员彻底抬脚完全松开油门的时候，动能回收刹车制动效果会达到最大，尽可能的回收电能。在驾驶过程中完全只靠一只脚踩油门，就可以轻松控制车速，抬脚就减速/踩下就加速，如果不是必要的红绿灯/堵车需要彻底刹停，几乎完全不用抬脚去踩刹车踏板。在市区驾车的体验上简直太完美，太舒服了。

  • 刹车制动：当然也不能完全不踩刹车，动能回收的最大制动效率是缓慢并且有限的。当你遇到紧急情况纯靠动能回收也无法及时刹车到安全车速的时候依然需要踩刹车踏板。不过特斯拉有个优点可以让你不用担心习惯动能回收后完全忘记踩刹车。特斯拉全车出厂就带有全套自动驾驶系统的传感器硬件，如果你没掏钱购买只不过是软件不开放而已，虽然没开放自动驾驶，但自动预警与低速下的紧急自动制动是依然存在的。当你高速的时候，前方车辆的距离根据当前车速差距如果系统判断出有风险，车内会有刺耳的蜂鸣与报警，并且在仪表盘把前方车辆红色预警闪烁，来通知司机当前速度有危险。而在很低速的情况下，如果遇到障碍车会停住。

  • 动力感受：很多人都说电动车开起来不像油车，更容易让人晕车，其实这是一个误区。晕车所产生的原因是因为人脑对车子起停速度的预期与实际动力效果不匹配产生的不舒适的感觉。电动车的实际动力来自于电机，而电机受油门踏板的幅度进行精细的电流控制。油门踏板=>电机动力这个过程是应该由汽车生产商进行精密调教的。调教的好了，油门踏板所产生的控制电流影响电机动力就可以非常的平滑，减少乘车人的晕眩感。调教的不好，踏板的幅度与力度，不能够平滑的控制电机动力，忽大忽小，就会增加乘车人的晕眩感。可以负责任的说，特斯拉在这块做的非常不错，完全不用担心会产生“电动车的晕眩感”

  • 方向盘控制：Model S 的方向盘控制功能也还是挺全面的，因为是怀档，档位控制杆在方向盘右侧，车灯控制与雨刷在侧的一个控制杆上，左侧还有一个小控制杆专门供自动驾驶使用。除了这些控制杆，方向盘左右两侧各有2个按键，方便进行仪表盘的展示切换以及音乐切换还有语音控制开关，可以方便的语音控制开启导航和接打电话。

  • 中控大屏幕：特斯拉的中控大屏幕真的很拉风，科技感超级强，几乎所有的汽车控制都可以在这里进行操作，并且特斯拉车上有个无限流量的联通卡，所以特斯拉的中控台是可以直接联网，并且有一个内置浏览器可以打开网页，不过这个浏览器内核估计是自己研发的，很多功能受限，无法播放视频和玩游戏 ╮(╯_╰)╭ ，就只是展示一些网站页面

    • Tips：当长时间处于联通信号不好的状态，比如地库内的时候，联通可能会短时间内关闭该数据流量卡的服务，12个小时之后再打开，如果紧急需要立刻开启，可以打特斯拉 400 电话，让客服中心帮你开启
    • 马一龙新的 Twitter 称：未来系统升级会升级成 Chromium，换句话说未来车内浏览器有可能会解锁更多娱乐能力

电量与续航

电动车最关心的就是续航和里程焦虑问题，我的 Model S 75D 是 75 度电，官网显示续航是475 km ，而这一点上各类新能源国产车纷纷打上 400km 甚至 450km 的里程，仿佛能和特斯拉一比高下。但实际上差距还是非常之大的，其原因就在于这续航里程的测定标准完全不同。

续航标准

电动车的续航受很多条件影响

  • 温度：影响电池的充放电效率，进而影响发电机效率，众所周知在低温环境下，电池都会面临很大问题

  • 载重：质量越大就越需要更大的动力去推动车辆前进，因此不同级别的车，他们的车身重量也能影响续航，只不过越重的车续航越少，车上载重量一样影响

  • 车速：开的越快自然耗能越大，高速的 120/130km/h 跟市区 60/70km/h 的速度比，续航自然差距大了去了（国产的拿些拿 60km/h 等速续航吹牛逼的厂商就是耍流氓）

  • 电耗控制：影响电量的因素主要因素里，除了发动机的前进动力占大头，还有车内空调的工作效率，以及动能回收的效率。前者费电，好的车辆会通过智能的温控系统尽可能的减少电量消耗。后者省电，在动能回收发生制动减速的同时可以回收一部分电能。

  • 风阻/轮毂：风阻风阻，顾名思义，阻力越小，耗电越少。而轮毂这玩意也会影响，轮毂越小，耗电越少。不过这两个因素都取决于整体车辆车身设计。

有这么多的因素可以影响电动车的续航，所以就要有一个续航标准来统一规范，实际上也是有好几种续航的标准存在

  • NEDC标准，欧盟的测试标准，四个市区工况加一个市郊工况为一个完整循环，速度曲线上尽可能在还原日常使用，但这种规律性极强的缓加速缓减速工况离真正的实际工况还是有差距的，所以得出的续航里程数据也会比实际偏高。

  • WLTP标准，低、中、高、超高速四个工况循环，速度曲线上做规律性不强的更复杂的变速运动，更接近日常使用。

  • EPA标准，美国环保署的标准，市区、高速、激烈驾驶、空调四种工况，比WLTP更接近日常驾驶，也更严苛。

以上三种标准，NEDC得出的续航里程会相对偏高，不严谨地推测的话，WLTP大概是NEDC的90%，EPA大概是NEDC的80%，这是不严谨地推测，准确数据肯定得官方公布。此外还存在第四种国内部分新能源厂商的”山寨”标准

  • “山寨”标准：等速续航，稳定在 60 km/h 的速度的行驶里程（太多国内厂商都拿这个指标来宣传续航，这就有点忽悠人了）

Model S 75D 真实路况续航感受

因为我在冬天刚买的车，还没感受过夏天20-30度户外温度下的续航真实情况，所以这里只介绍一下冬天里，我的真实用车感受。

  • 里程预估区间

我的是 75D 标准续航，EPA标准下续航里程为 475km。为了避免电池频繁完全充满所造成的电池损伤，特斯拉推荐非远途情况下充电设置上限保护。因此我设置的充电上限是 90%。现在的特斯拉都支持国标充电口，所以国家电网的直流交流充电桩都能用，不是必须去特斯拉超充，所以我没有太严重的电量焦虑，一般留15%开始找充电桩都非常来得及。再额外多打点富裕量，里程不焦虑的电量区间为 20%-90%。

  • 真实路况体验

特斯拉可以查看最近 瞬时/10km/20km/50km 的平均电耗曲线，这个电耗什么概念呢？132 Wh/km 代表每公里要消耗 132 Wh(瓦时) 的电，而 1000 Wh 就是 1度电～换句话说，132的电耗，就等于百公里13.2度电。1度电的电钱6毛～2块不等，也就是说百公里 8块-26块不。前提是路况够好能够开出132的电耗，一般情况下冬天市区内 150-250不等，看堵车。关于充电价格与感受，后面会详细介绍。

再说说我的从国贸到回龙观霍营大概刚好 30km 的上下班路程的实际体验，说句实话这个真实路况其实挺堵车的，国贸-农展馆-三元桥-西坝河-马甸-健翔桥-八高收费站，都是比较赌的地方，直到进了八高收费站，才会有所改善。全程电耗都是按照 25km平均来算的，上面那个 132 其实是一段 10km 平均的优质路况短时电耗拿来拍照装B的～

  1. 刚开上电动车，脚下没轻没重，喜欢猛加速感受踹背感，我开出过 260 的电耗，全程开暖气15% 的电
  2. 依然是脚下没轻没重，但是尝试不开暖气开座/椅方向盘加热，全程 12% 的电
  3. 逐渐习惯正常驾驶，大概维持在 200 左右的电耗，全程开暖气 10% 的电
  4. 春节回来天气暖和一点，可以不开暖气也不冷了，大概维持在 170 左右的电耗，全程 8% 的电

总体感受就是冬季电动车表现确实不佳，加上室内路况非常拥堵，追求舒适打开空调的情况下，10%就跑 300km 了，而 EPA 美国标准下的续航是 475km。

  • 导航电量预测

特斯拉的车载导航前面介绍过优点，可以在仪表盘显示实时导航，关于导航电量的预估上，特斯拉的电量计算也是非常准确的，能够极大程度的减少电量焦虑问题。

每次使用特斯拉的导航，导航都会显示单程去预计消耗的电量，以及预计到达的时间。同时还会贴心的帮你计算好往返的电量。并且他的电量能耗计算应该也是根据你的驾驶习惯以及历史的电耗水平计算出来的，可以说算的是非常非常的准。并且它也会根据驾驶状况来实时更新电量预测，如果路况好了，目的地剩余电量预测还会涨，如果路况不佳堵车严重，则会降。在我开了一个多月后，从国贸到回龙观霍营这大概一个半小时的路程规划上，基本上出发预估消耗 8%，那么最终到达目的地后基本上也就是消耗 8%，上下浮动不会超过1%。在开去任何别的目的地，电量预估大多数情况下依然都能保持在 1% 的浮动误差内。

换句话说，只要提前规划好，在我出发前确认到达目的地的剩余电量足够，往返情况也符合预先规划，那么我就完全可以放心大胆的开出去。完全不用担心出发前预计你有电量能到达，结果半路用光的情况。

  • 静止驻车耗电

特斯拉完全静止驻车状态下也是会耗电的，官方有个宣传视频里介绍说，一般情况下驻车一整天耗电 3% 左右。因为我春节前提的车，春节去台湾玩8天，我刚好借此关注了一下驻车耗电的情况，真实感受差不多，大约每天耗电 2% 多一点

充电注意事项

无论是交流慢充，还是直流快充，还是特斯拉超充，都不会损伤电池，造成电池的续航下降！

无论是交流慢充，还是直流快充，还是特斯拉超充，都不会损伤电池，造成电池的续航下降！

交流本身是慢充没事，而直流快充/特斯拉超充，他们的充电曲线都是浮动的，只有在电池比较缺电的情况下会进行大功率充电，电池逐渐接近满电量的情况下，他们的输出功率会下降到对电池无损害的充电效率。真正对电池有损害的是，在特斯拉系统电量低于0%后，还继续行驶，特斯拉系统电量 0% 依然能行驶十几公里，但这个过程会造成电池损伤。另外还有频繁多次充电量到100%，如果要行驶长途一两次充到100%没问题，每天晚上插电桩都充100%就会损伤电池。

推荐的充电方式：

  • 如果拥有固定生活区的充电桩：每次都充满，无论电剩余多少，插上电就到充到保护电量上限即可，这样基本上只使用顶部电量～
  • 如果是公用电桩：尽量多用电量，剩余电量比较低再去充电，充电充到70%-80%即可，这样基本上只使用底部电量
    • 这样的好处不是为了保护电池，而单纯是为了节省时间，每次只有底部电量充电效率才最好，越到满电量，充电越慢。对于公共电桩来说，往往最珍贵的是专程去充电的时间与停车费。

自动驾驶体验

特斯拉的自动驾驶所需要的硬件，都是随车已经加装好了的，哪怕你订车的时候不选购自动驾驶套件，他的硬件各大传感器摄像头，都已经在你的车上了。但是想要打开这个自动驾驶功能需要付费然后软件开启。所以允许你在订车的时候不选购自动驾驶，以后再补买。特斯拉把所有自动驾驶功能分成了3个产品包

  • AutoPilot：2019年2月28日以后才能选购，3w7
    • 自适应巡航
    • 道路保持
  • 增强 AutoPilot：2月28日以后无法购买，4w6
    • 自适应巡航
    • 道路保持
    • 自动变道
    • 自动泊车
    • 召唤（召唤出车位）
  • FSD 全自动驾驶：2月28日以后才可以选购，6w5 （需要先购买 3w7 的 AP）
    • 自适应巡航
    • 道路保持
    • 自动变道
    • 自动泊车
    • 召唤
    • 自动进出高速，自动超车并线（当前不提供，后续随车辆系统OTA升级获得）
    • 高级召唤，车随人走（当前不提供，后续随车辆系统OTA升级获得）
    • 识别交通灯和停车线（当前不提供，后续随车辆系统OTA升级获得）
    • 全自动城市驾驶（当前不提供，后续随车辆系统OTA升级获得）

早些时候，所有车都只能买增强AP。但是2019.2.28之后，增强AP停售，拆成了 “阉割版”AP 和 “预售版”FSD。可以看出来 FSD 虽然号称全自动驾驶，但是主要增加的三个功能都现在还没做好，后续好了才会推送给车辆进行 OTA 升级，所以算是一种预售。而 AP 就纯粹是阉割了。

我的自动驾驶体验

我当初提车的时候一律按着最低配来，毕竟本身选 S 已经超出了我的预算很多，就打算以后再补 票 增强Ap（当时还是卖增强Ap的）。但我刚提车开了没两三天，突然弹出个30天试用邀请函，一股浓浓的互联网风，我就很开心的试用起来，“真香”！

都说 AP 最好的使用场景是在高速上，什么自适应巡航，自动变道，自动变道并线，都是高速上才方便感受的，但这 30 天内，刚好过春节假期要出去玩，最后我也没找到机会在高速上试验一把，所以就只在城市内，在中低速下感受了一把，自适应巡航/道路保持/自动变道/自动泊车/召唤。直到老车主补偿计划一出来，趁着比较大幅度的优惠，彻底购买了“预售版” FSD 。

  • 自适应巡航：任何驾驶状态，向自己波动 Ap 控制杆一次，即可开启
    • 解放了脚，但还得控制方向盘
    • 缓慢可预期减速制动还算柔软舒适
    • 堵车状态下的走走停停，刹车很硬，不如人踩的柔软，可能容易让人晕车
    • 前后车距保持得有点远，堵车状态，很容易被人插队进来
    • 插队的人如果粗暴一些容易判断不及时发生刮蹭，还是小心为妙及时人工介入；
  • 道路保持：在自适应巡航状态，再次向自己波动 Ap 控制杆两次，即可开启
    • 解放了双手，方向盘交由 Ap 控制
    • 保持在道路线中心，可以跟随道路自动转弯
    • 前方没车的话，会加速到道路限速
    • 前方有车的话，会保持安全车距跟车
    • 双手不能长时间离开方向盘，否则会报警
    • 长时间报警会还不手扶，车辆会判断驾驶员遇到问题，直接逐步减速停车开双闪
  • 自动变道：在道路保持状态下，打转向灯，会自动驾驶变道，可以实现超车
    • 目标车道后方无车的情况下，变道比较平稳顺利
    • 目标车道后方有车的情况下，自动变道判断会比较犹豫，迟迟不并
  • 自动泊车：好多高档车应该都有
    • 当挂到R档进行倒车的时候，如果传感器探测到有车位存在即可开启
    • 支持侧方停车/竖直停车
    • 探测车位的条件是要识别到车位两旁有车，只是地上画车位线但两侧没车就无法识别
    • 我的固定车位有点小，两侧车停稍微靠近点，就无法识别出来，只好手动倒进去
    • 两侧车停的没那么规矩的时候，可能影响对车位的判断，导致识别的车位歪斜，如果任凭他自己泊车，会发生刮蹭（我看了一眼觉得不对，没敢点自动，手动停的）
    • 微博CEO来去之间的车，也被自动泊车坑过刮蹭了
  • 召唤：用钥匙或者手机 App 都可以实现，驾驶员在车外，遥控车辆入库出库
    • 打开召唤功能设置，关闭「指定长按启用」
    • 当车位非常狭窄，停好车后两侧车门都无法打开下车的情况下
    • 直线开出车位，双击P档进入召唤入库模式，驾驶员离开关闭车门后，自动后退进入车位
    • 也可以直接通过钥匙召唤车辆支线进出车位
    • 开启召唤模式（App点召唤，或者长按钥匙顶部5秒）
    • 前/后 两个按钮控制车辆前进后退，可以自动躲避障碍物

“预售版” FSD 尚未提供的功能，等待之后特斯拉远程推送系统升级即可支持，并且我的 75D 还享受免费加装更多感应 AI 芯片的权利，等待 FSD 推送，开去交付中心升级硬件即可

远途旅行体验

其实在网上看到很多 Tesla 车主进行远距离旅途，可以沿路规划充电路线，然后长距离跋涉，我也很想尝试尝试。

  • 特斯拉的车载导航直接输入远途目的地，会自动规划路上的充电路线
    • 规划出来的路线只会选取特斯拉超级充电桩
    • 超充一般都在室内，所以规划充电，会导致进出高速，进入室内
  • 国内首席特吹“小特叔叔”，做了一款 Tesla 专用 App，用它规划远途充电路线
    • 支持超级充电，国家电网，特来电，星星充电等几种电桩
    • 支持像 Tesla 专属 App 一样控制车辆，以及 apple watch 等车辆控制支持
  • 百度地图，支持在导航路线上的充电桩显示
    • 支持超级充电，国家电网，特来电，特斯拉几种电桩
    • 并不支持专门通过充电桩规划路线，只是显示途径电桩

这不目前天气还比较冷，等天气暖和了，电池给力了，必须长途跋涉一番，感受一把长距离“电动爹”旅途。

  • 初级难度：开车去雄安参观转一圈，当天去当天回～
  • 中级难度：开车去天津，再去塘沽，随便转转玩玩～
  • 高级难度：开车去上海，感受一把～
