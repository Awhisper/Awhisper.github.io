特斯拉电动爹的亲密接触

发表于 2019-03-10 | 分类于 生活随笔 | | 阅读次数:

据说特斯拉入华的时候，曾经申请过一个叫“拓速乐”的商标，虽然后来又更名成了特斯拉，但是这个“拓速乐”三个字总是让人充满喜感 ╮(╯_╰)╭
晒一下我的这辆 “拓速乐”

北京新能源指标

由于北京燃油车指标摇号常年不中，已经积累到了 5 倍概率，依然看不到中号的希望，几千分之一即便乘以 5 依然还是遥不可及，于是我在 2016 年中切换成了北京新能源指标，终于在一年多之后自然的“排队”排到了新能源号牌，号牌有效期1年。

18年上半年的北京车展也特意去逛了新能源专场，虽然 Tesla 看起来是压根不屑于参加北京车展，没看到展台，但也还是了解了非常多的其他厂家的。由于北京的新能源牌照只认纯电不认混合，在纯电的厂商里，当时看起来比较好的就是“比亚迪”和“荣威”了。另外还看到很多北汽的超低价5W-7W不等的占号车。

18年下半年，我也专门走访了比亚迪，宝马，特斯拉的销售门店，现场深入摸摸车深入的进行了解.

我当时的电动车选择心态是

  • 北汽垃圾车：
    • 死也不想买～
    • 6W块钱左右基本等于白扔，用车那个里程与续航就是扯淡，就算不用还得天天发愁停车，我还不如每年花 2000 多左右，去淘宝办理指标延期
    • 我对北汽这个在北京地方保护政策下的企业充满了鄙视，觉得这种企业就该死，就是靠吸国家补贴的蛀虫，更何况现在国家补贴逐渐在减少
  • 新能源创新车企：
    • 在18年的北京车展上，那会 蔚来/小鹏 什么的好像都没怎么看到 ～
    • 对外的宣传也不多，18年上半年的时候完全没概念，似乎下半年才突然冒出来很多
  • 比亚迪/荣威：
    • 还是挺有吸引力的，但还是不太想买～
    • 价位 10-40W 的各个区间都有，各方面感觉看起来还是挺不错的
    • 北京号牌如此之艰难，车买下来想买个稍微好点的，想上个 BBA 级别中档点的
  • 宝马:
    • BBA 中唯一有纯电车的，i3/i8～
    • i8 那接近 200W 的价格就算了吧 =。=
    • i3 伴随着 4S 店的各种优惠之下，能用 30+ 拿下
    • i3 实在是太丑了，丑到不能忍，虽然是2排座位，但空间看着感觉没比 QQ 大多少
    • 宝马 530 的混动真的很吸引人，但是北京的新能源不认混动，没辙
  • 特斯拉/拓速乐：
    • Model S/X 的价位一下超出宝马 530Le 不少，预算压力还是挺大的
    • Model 3 之前一直是预定不知道猴年马月能拿车，但价格非常有吸引力
    • Model S/X 太好看了尤其是 S，但 3 的外形就觉得有一点丑了
    • 我对 SUV 车型没那么感冒，城市通行还是想要舒适的轿车
    • 去销售门店试驾，感受驾驶体验，无论是车辆召唤，动能回收，还是自动驾驶，感觉真的是太好了，欲罢不能

从上面对于纯电动车的选择心态上可以看出来，我已经非常非常偏向特斯拉了，不过我还是有个看法，我不是啥特斯拉吹，或者电动吹，我始终认为如果新能源指标能买混动，绝对死也不买电动爹，但我身在北京，就是这么的无奈，唉～

曲折的下订单心路历程

因为生活与工作的原因，2018 年下半年，我换到了国贸上班，从我自己家回龙观到国贸的通勤变得很不方便。买车的心思越来越重，BBA 的 4S 店我也没少逛，在确认宝马奔驰奥迪的纯电动车新车型遥遥无期，毫无可盼的情况下，就开始更高频率的关注特斯拉的各方面新闻，也专门跑去特斯拉的直营销售店去试驾与体验。

本来 Model S 的最低配 75D，大概 80W 能下来，就是一直觉得超出预算挺多的，一直没下定决心。

2018年7月，突如其来的贸易战让美国进口车关税从25%提升到40%，于是特斯拉的价格直接涨了20W，丐版 Model S 也得 100W 才能拿下，彻底浇灭了我蠢蠢欲动的欲望。

2018年11月，随着贸易战你来我往打了那么久，似乎 Elon Musk 马一龙老铁察觉到了贸易战即将松动缓和的信号，官方宣布补贴关税差价导致的价格上涨，75D 又回到了 80W 能拿下来的价格。并且官方调整了配置清单，把皮质座椅/全景天窗/木制内饰材质/高级音响系统/生物武器防御级别的空气净化等全都做成了标配不在加价选配，相当于加量不家价，更加的划算了一些，于是我又开始蠢蠢欲动了。

2018年12月，贸易战在汽车关税上，两国政府宣布进入4个月的停战期，关税恢复正常水平到3月31日（这个时间很关键），马一龙老铁又在原本官方补贴关税，不到 80W 左右拿下 75D 的基础之上，又降价了几万块，标配裸车 74W 左右拿下 Model S 75D，我心里更加的躁动了。

并且18年底，传说中的 Model 3 已经离进入中国越来越近了，官方全面开放了后驱长续航/双电机/双电机性能版三款车型，并且销售统一口径，3月底肯定能提现车，现在可以赶紧下订单。3月底这个时间恰恰正是关税停战期的 Deadline ～ 并且放出了最新的消息，特斯拉要在上海建立工厂，专门组装生产 Model 3（不产S/X）～ 不过 Model 3 现在公布的这三款下来，算上购置税等落地也得 50-70W 不等看配置，真正让 Model 3 值得期待的是还没面世，就是 3.5W 美元那款低配，上海工厂生产的话估计能进 30W 的那款。而且 Model 3 亲自去看了一下，空间真的比 S 小了很多，没有仪表盘感觉特别特别的不习惯。

2019年1月初，马一龙老铁在推特上又一鸣惊人了，说从 1.14 周一起停售 75D，就只有高配 100D 了卖了，得活脱脱贵出 10 多W。这尼玛我一下坐不住了，之前心动 S 已经很久了就是实在是有点超出预算，因此一直在犹豫，这一下好了，停售就彻彻底底的更加超出预算彻底没戏了。

头像是新垣结衣的销售小姐姐赶紧在给我计算更加便宜划算的买法。现在因为 Model S 如果是预定车一定会标配皮质座椅/木制内饰材质，不用额外选配加钱。但现在已经入关的现车里刚好就有，半皮半织物座椅+黑色烤漆内饰的现车，并且是我最想要的红色车漆。（说真的烤漆内饰简直是意外惊喜，因为车内的木制装饰颜色看起来太老气了，换成烤漆反而感觉更加好看的多）

  • 皮质座椅/木制内饰本为标配，因此买这台现车反而会给我优惠 -35500
  • 因为选的红色车漆需要 +23500

这一加一减，销售小姐姐算了一下 73W 能拿下我最想要的红色 Model S ，我这个心动啊，我这个纠结啊，最终下定决心，在 1.18 的这个周五，直接交了定金下了订单。而下一周就停售了（事实证明，马一龙说话也有扯淡的时候，现在 75D 改了个名字又 TM 重新开卖了）

令人惊讶的提车速度

本来 1.11 下的订单，按着国内四儿子店的尿性，我心想怎么不得 2.10 春节假期后提车来。谁成想特斯拉这种没有4S的直营销售模式，公司要求现车必须在5哥工作日内交付，提车速度快到令人发指！我刚好下一周在杭州出差，谁成想一周的时间就可以提车了。。。

我的新能源资格本来在 2019.02.26 到期，但我确实没想到能这么快突然就下定决心订车，也没想到特斯拉提车这么快速，我早在12月就办好了电动车延期，淘宝花了2300代办。这下这个延期彻底白办了 ╮(╯_╰)╭

  • 1.11 周五晚上下的订单
  • 1.12 听说我周日要飞机飞杭州，在周六就联系好招行的贷款资质认证
  • 1.15 招行车贷批准通过，在杭州转账，结清首付尾款，缴纳人保保险
  • 1.16 招行车贷到账，我可以提车了，然而我人还在杭州
  • 1.19 周六，我刚从杭州回来就能提车了，临牌也办好了
  • 1.23 周三，请了天假上好了车牌

特斯拉交付中心在亦庄，我本来计划在大清早约朋友赶往亦庄提车，但是交付中心的小姐姐说，现在国内也在推广美国的那种送车上门的提车模式，我可以成为首批体验这种服务的车主～那必须体验啊～周末中午，交付小姐姐开着我的红色 Model S 送车上门啦～

特斯拉还赠送了全套上牌服务，帮我提前办理各种上牌手续，只等一天亲自跑到五方桥现场验车选牌。

北京新能源的号牌允许在网上选号，可以自己输入自己喜欢的号进行尝试，因为新能源号牌还很空，应该能比较容易的选到心仪的号码。

但是网上选好不一定当天能出，可能还要再拖一天来拿号牌上牌。之前的年假已经被我去日本用的快光了，所以额外多请一天假对我来说太奢侈了，硬着头皮现场摇号吧！

随机五十个号选一，走你！一发入魂！

本来看着这随机的五十个号我正看花眼呢，周围围观的群众一眼就帮我指出来这里面看着最牛逼的号，AD29800，都在说我这手气简直太好了，得嘞，就它了！

驾驶与操控体验

电动车在驾驶的感觉上真的和以前开燃油车的感觉太不一样了，简直太舒服太好开了～

  • 线性加速：电动发动机的加速是线性的，油门轻轻踩下去就能直接有很明显的速度感。不像燃油发动机那样用力踩下去转速彪到3～4迁，速度也是逐渐加起来的那种感觉。所以在堵车路段，电动车快速起步钻进钻出有很大的优势，并且不用担心为了快速起步踩重油门造成的高转速导致速度起来后不好控制。

  • 推背感：Model S 75D 的百公里加速是 4.4 秒，如果上 P100D 能到 2.9 秒。查了一下官网，保时捷 Boxster 是 5.6 秒，Boxster GTS 4.1 秒。不扯那么多数字，就说一下真实感受，推背感这个词可以换一个字，“踹”背感 ╮(╯_╰)╭ 夜深人静找个完全没人的路上，一脚“电”门踩到底，一发地板“电”，那个感觉简直太夸张了，把你按在座椅上的加速感～顺带说一下，电动发动机在加速这快仿佛也有着很大的优势，很多很多的国内新能源车型都能把百公里加速这个指标拉到 5 秒以内。

  • 动能回收：动能回收的驾驶体验简直太好了！所谓动能回收就是，当驾驶员轻抬油门的时候，就会自动触发刹车制动，并且回收一部分电能。而当驾驶员彻底抬脚完全松开油门的时候，动能回收刹车制动效果会达到最大，尽可能的回收电能。在驾驶过程中完全只靠一只脚踩油门，就可以轻松控制车速，抬脚就减速/踩下就加速，如果不是必要的红绿灯/堵车需要彻底刹停，几乎完全不用抬脚去踩刹车踏板。在市区驾车的体验上简直太完美，太舒服了。

  • 刹车制动：当然也不能完全不踩刹车，动能回收的最大制动效率是缓慢并且有限的。当你遇到紧急情况纯靠动能回收也无法及时刹车到安全车速的时候依然需要踩刹车踏板。不过特斯拉有个优点可以让你不用担心习惯动能回收后完全忘记踩刹车。特斯拉全车出厂就带有全套自动驾驶系统的传感器硬件，如果你没掏钱购买只不过是软件不开放而已，虽然没开放自动驾驶，但自动预警与低速下的紧急自动制动是依然存在的。当你高速的时候，前方车辆的距离根据当前车速差距如果系统判断出有风险，车内会有刺耳的蜂鸣与报警，并且在仪表盘把前方车辆红色预警闪烁，来通知司机当前速度有危险。而在很低速的情况下，如果遇到障碍车会停住。

  • 方向盘控制：Model S 的方向盘控制功能也还是挺全面的，因为是怀档，档位控制杆在方向盘右侧，车灯控制与雨刷在侧的一个控制杆上，左侧还有一个小控制杆专门供自动驾驶使用。除了这些控制杆，方向盘左右两侧各有2个按键，方便进行仪表盘的展示切换以及音乐切换还有语音控制开关，可以方便的语音控制开启导航和接打电话。

  • 中控大屏幕：特斯拉的中控大屏幕真的很拉风，科技感超级强，几乎所有的汽车控制都可以在这里进行操作，并且特斯拉车上有个无限流量的联通卡，所以特斯拉的中控台是可以直接联网，并且有一个内置浏览器可以打开网页，不过这个浏览器内核很多功能受限，无法播放视频和玩游戏 ╮(╯_╰)╭ 具体大家看看中控面板展开后的大图吧～

  • 导航：特斯拉的导航有个显著的优点！中控大地图的线路导航也会同步到方向盘下方的仪表盘上，也就是说看导航压根不需要斜眼看中控，直接看仪表盘就可以非常方便的看到实时导航的每个细节，这个要好评一下！不过槽点紧跟着就来了，特斯拉的导航在中国居然是腾讯地图！实时路况堵车情况只有大一点的主干道才有显示，很多便一些的小路根本没有实时路况，从而计算的躲避拥堵的路线是不准的！甚至有一些更偏的小路地图上根本没有！下图是仪表盘的导航～

  • 远程控制：

    • 手机：特斯拉有个 App ，随时可以通过车上的移动网络查看车子的状况，包括运行状况，电量状况，充电完成的推送。也可以远程操作车灯/鸣笛/前后备箱，也可以提前打开空调or座椅加热，提前让车内温度处于舒适状态。如果出门忘带车钥匙，无法开车，也可以通过手机 App 解锁车辆进行驾驶，只不过要小心，如果车停在信号不好的地方，比如地下车库，那么 App 就无法解锁车辆了。
    • 钥匙：钥匙的造型就是一个小车模，靠近车身会自动解锁，离开车身会自动落锁，比较方便，还可以通过钥匙上的召唤功能，近距离控制车子自己驶出车位。

  • 行车记录仪：特斯拉的车头前方内置了一个高清摄像头，整个自动驾驶系统中的一个核心识别硬件，这个摄像头也可以顺带用来当作行车记录仪来使用。但特斯拉官方不提供存储设备需要你来提供一个 U 盘开启内置行车记录仪功能。具体操作是自己准备一个 U 盘并格式化成 Fat32 系统（不要超过 32G 的 U 盘，不然就算格式化成 Fat32 也容易导致无法识别），在 U 盘中创建一个叫 TeslaCam 的文件夹，然后再插入车上的 USB 口上，就可以被识别到并且开启行车记录仪功能。

  • 彩蛋：特喵的中控大屏幕上有一个彩蛋专区，在关于面板里下拉就可以打开一大堆彩蛋程序，有一堆乱七八糟的玩意，而且还随着整车系统的升级还在更新更多的玩意，看一眼就知道了～

    • 有放屁座椅功能，可以在其他座椅的位置上发出放屁的声音
    • 有涂鸦板，便签板，可以记录东西
    • 有把大屏幕地图导航变成火星地面+火星探路者的样子
    • 有个脑残的篝火，打开后屏幕变成了一个篝火堆动画，驻车的时候为了省电，开这玩意让人看着觉得暖和一点，进行心里安慰 ╮(╯_╰)╭
